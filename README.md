# Briefly - Daily Article Digest

A modern web application for subscribing to daily article digests via SMS or email.

## Frontend

Built with Next.js 14 and Tailwind CSS.

## Getting Started

Frontend Setup
```bash
# Install dependencies
cd frontend
npm install

# Run development server
npm run dev
```

Database Setup
```bash
# Database Setup
npm run db:up

# Run database migratiosn
npm run db:migrate

#Seed local data
npm run db:seed

# Stop with
npm run db:down
```

Backend Setup
```bash
# Install dependencies
cd frontend
npm install

# Run development server
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
